Handwritten Character Recognition

Objective

The objective of this project is to recognize handwritten digits using Deep Learning and Convolutional Neural Networks (CNN). The model is trained on the MNIST dataset to accurately classify handwritten digits from 0 to 9.

Dataset

MNIST Handwritten Digits Dataset

- Training Images: 60,000
- Testing Images: 10,000
- Image Size: 28 × 28 Pixels
- Classes: 10 (Digits 0–9)

Technologies Used

- Python
- TensorFlow
- Keras
- NumPy
- Google Colab

Project Workflow

1. Load the MNIST Dataset
2. Preprocess and Normalize Images
3. Build a CNN Model
4. Train the Model
5. Evaluate Performance
6. Predict Handwritten Digits

Model Architecture

- Conv2D Layer (32 Filters)
- MaxPooling2D Layer
- Flatten Layer
- Dense Layer (128 Neurons)
- Output Layer (10 Classes with Softmax Activation)

Results

- Model: Convolutional Neural Network (CNN)
- Dataset: MNIST Handwritten Digits Dataset
- Accuracy: 98.57%
- Loss: 0.0457

Conclusion

The CNN model successfully achieved an accuracy of 98.57% on the MNIST dataset. The low loss value of 0.0457 indicates effective learning and strong performance in handwritten digit recognition. This project demonstrates the power of Deep Learning and CNNs in image classification tasks.
